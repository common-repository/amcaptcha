== Description ==

Provided by [Multitoolbox](http://multitoolbox.com/).

Simple, but strong captcha, built to **prevent spam in Wordpress comments**. Succesfully blocks Xrumer-like tools.

To leave a comment on your blog, visitors need to simple check the captcha's field. That's all! Forget about annoying typeing of letters and digits.

**Features:**

* Extremely simple passing for human;
* Built-in id's and function name's generator to produce unique and random values, Xrumer won't pass;
* Catchy look, you commentators won't miss;
* Captcha's code will be pasted into the comment form via JavaScript, so spam-bots will get a big parsing problems; 
* Comment text saving, so if commentators forgot to check captcha's field, they never have to write commen again.

**Languages:**

**Amcaptcha is bulletproof solution tested on many blogs.**


== Installation ==

To set the plugin up, you need to:

1. Upload `amcaptcha.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

That's all!

== Frequently Asked Questions ==

= What if my visitors has javascript disabled in their browsers? =
Amcaptcha lets them know about it and kindly asks to enable this feature.

= How to uninstall Amcaptcha? =
Simple deactivate it through the 'Plugins' menu in Wordpress.
